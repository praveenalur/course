Business Case 1: Sorting Public Transports by Daily Ridership

Problem Statement

Sort a list of transport modes by daily ridership using the Merge Sort algorithm.

Challenges

  • Sorting a custom data structure (Transport class).
  • Efficient sorting with optimal time complexity.
  • Maintaining stability in sorting.
  • Managing memory for temporary arrays during merging.

Solution

  • Use the Merge Sort algorithm for efficient sorting.
  • Implement a custom Transport class to hold data.
  • Use temporary arrays (leftArr, rightArr) for merging sorted subarrays.

Algorithm Used

Merge Sort: A divide-and-conquer approach that splits and merges subarrays to achieve sorted order.

Data Structures Used

  • Class (Transport): Stores transport data.
  • Vector (std::vector): Stores a list of Transport objects and temporary arrays for merging.

Efficiency

  • Time Complexity: O(nlogn)
  • Space Complexity: O(n) due to temporary arrays.
